Cloud Study Jam 參加心得

2021-08-13 | 閱讀約1分鐘 | 分類 cloud, GCP

Cloud Study Jam 是由 Google 開發者團隊所發起的 Google Cloud 技術培訓計畫，主要邀請台灣與港澳的開發人員學習 Google Cloud 的相關技術

前言

參加此計畫除了能學習 Google Cloud 的相關技術，也可以獲得:

  • 免費的 Qwiklabs (免費使用期限為30天) 訂閱來學習 Google Cloud Platform 技術
  • Google Cloud 技能徽章 (Skills Badges)，可以添加在個人履歷中
  • 精美的活動禮品

活動日期:  2021.6.1 - 2021.6.30

活動官網:  Google Cloud 開發者技術培訓計畫

Qwiklabs 網站:  Qwiklabs

禮品取得資格

註冊這個 Cloud Study Jam 的活動且完成活動指定的 Quests，就能獲得禮品

  • 完成 6 個以上活動指定的 Quests (i.e. 取得 6+ badges)，即可獲得精美 Google Cloud T-shirt 一件
  • 完成 10 個以上活動指定的 Quests (i.e. 取得 10+ badges)，即可獲得精美 Google Cloud T-shirt 一件以及 Google Cloud 後背包一個

參與活動心得

在這段期間，我在 Qwiklabs 上學習了很多 GCP 上的操作技能以及開發工具，在這雲端平台上可以完成許多繁雜的工作，像是架設一台 VM 、自動化部署網站或是測試 ML 模型…等，都能夠簡單又快速地解決我們的煩惱，只要照著 GCP 上提供的詳細步驟及說明，就可以順利完成任務，有時候遇到不懂的操作，可以到粉專社團上發問，通常會有大神幫你解惑，又或是網路上 google 一下，會找到很多教學文章可以幫忙解決困難。

另外，這個活動還提供了 30 天免費的 Qwiklabs 訂閱，讓我可以利用這個訂閱使用 Qwiklabs 上所有的學習資源（也就是吃到飽的意思），只要不超出它的一些限制條件，都可以時時刻刻在 GCP 上操作與學習，更重要的是，除了學習提升自己的技能之外，還能拿到很棒的禮品。(本篇重點 XD)

結語

之前我只有聽過「雲端」這個名詞，但沒有實際接觸過，而經過這次活動後，讓我更加了解雲端以及雲端平台，這次活動真的學到很多東西，希望未來 Google 團隊能夠多多舉辦類似的學習活動，讓大家能夠互相學習、分享經驗。
