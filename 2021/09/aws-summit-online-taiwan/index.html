前言

這次參加 AWS 台灣雲端高峰會除了能觀看豐富的議程外，也能參與 AWS 以及合作廠商的抽獎活動和奪牌活動，甚至還有實戰工作坊可以練習

活動時間:  2021.09.02  9:00 am ~ 4:30 pm

活動官網:  AWS 台灣雲端高峰會

奪牌任務活動規則

在活動時間內，觀看議程收集獎牌，AWS 官方有提供 「任務說明」，只要完成任務就可獲得對應的獎牌


查看下方「任務達成卡」是否有獎牌成功連線為水平線、垂直線或對角線：

  • 若連成 1 條線即獲得 AWS 隨行充電鑰匙圈 1 組
  • 若連成 2 條線 (或以上) 即獲得 AWS 無線充電滑鼠墊 1 個

參與活動心得

從早上到下午觀看了許多議程，從中真的學到了許多 AWS 的創新應用與基礎架構平台，我印象最深的是 AWS 的 SageMaker，它能讓使用者可以輕鬆建立、訓練、偵錯、部署並監控機器學習模型，並專注於開發機器學習模型，而非環境的設置與開發工具的轉換。另外，Intel 的 OpenVino 也讓我印象深刻，OpenVino 是 Intel 開發的電腦視覺與深度學習應用的開發套件，具有模型優化器和推理引擎，可支援多種作業系統，以及常見的深度學習框架 Caffe、TensorFlow、Mxnet、ONNX 等所訓練好的模型與參數，結合自家硬體加速晶片，在 AWS Greengrass 上使用 OpenVino 引擎的話，能在邊緣運算上達到不錯的效果

除了學到新知識與技巧外，這次活動也提供了很多豐富的獎品，在會後填寫整體活動的會後問卷還能獲得 25 美元 的 AWS Credit 服務抵用券，全程活動都是免費的，真的非常物超所值~~
